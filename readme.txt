== Description ==

Easily calculate and log your workout routines from popular strength training programs or create your own.

== Installation ==

1. Unpack `warmupreps.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What are the shortcodes that this plugin uses? =

[show_all_workouts] displays the user interface.
